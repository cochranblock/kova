# Kova Theme — Logo-Aligned

**No gamification.** No XP, level, streak, achievements. Keep the coolness.

---

## Visual Identity (from logo)

- **Palette:** Neon electric blue, teal, purple/magenta gradient
- **Background:** Dark cosmic (near-black, subtle nebula)
- **Motifs:** Circuitry, neural network, data flow
- **Feel:** High-tech, dynamic, emissive
